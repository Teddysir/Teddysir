- 지식 공유를 즐기며 **동아리 회장 및 SSAFY 프로젝트 팀장**으로서 팀의 목표를 달성해 왔습니다. 교육 봉사를 통해 다진 **원활한 커뮤니케이션 스킬**을 바탕으로 동료와 기술적 시너지를 내는 개발자로 나아갑니다.
